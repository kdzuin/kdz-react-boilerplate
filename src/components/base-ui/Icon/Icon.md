### Basic Usage

```jsx harmony
<Icon name="icon-pencil" />
```

### Sizes

**Icon.Size.DEFAULT**
```jsx harmony
<Icon 
    name="icon-pencil"
    size={Icon.Size.DEFAULT} 
/>
```

**Icon.Size.SMALL**
```jsx harmony
<Icon 
    name="icon-pencil"
    size={Icon.Size.SMALL} 
/>
```

**Icon.Size.LARGE**
```jsx harmony
<Icon 
    name="icon-pencil"
    size={Icon.Size.LARGE} 
/>
```

**Icon.Size.STRETCH**, container is 200x200
```jsx harmony
<div style={{
    width: '200px', 
    height: '200px', 
    border: '1px solid #ccc'
}}>
    <Icon 
        name="icon-pencil"
        size={Icon.Size.STRETCH} 
    />
</div>
```
